# SNDBManager
数据库管理

## Features

- 对FMDB的高度封装

## Usage

更多使用参见[SNDBManager.h](https://github.com/snlo/SNDBManager/blob/master/SNDBManager/SNDBManager/SNDBManager.h)

```
import <SNDBManager.h>
```

## Problems

- 暂时基于FMDB
- 需要查阅[SQlite语法](http://www.runoob.com/sqlite/sqlite-alias.html)

## Installation

```
pod 'SNDBManager'
```

## Requirements

iOS 8.0 或者更高版本
